Install Prisma 7 dependencies into apps/api and write the complete schema.

**Step 1: Add Prisma dependencies to apps/api/package.json** — add to dependencies: `"prisma": "^7.0.0"`, `"@prisma/client": "^7.0.0"`. Then run `pnpm install` from root.

**Step 2: Initialize Prisma** — from `apps/api/`:
```bash
cd /home/alfie/bite-byte/apps/api && npx prisma init --datasource-provider postgresql
```
This creates `apps/api/prisma/schema.prisma` and `apps/api/.env`.

**Step 3: Update apps/api/.env** to match Docker Compose credentials:
```
DATABASE_URL="postgresql://bitebyte:bitebyte@localhost:5432/bitebyte_dev"
```

**Step 4: Replace apps/api/prisma/schema.prisma** with the complete schema from RESEARCH.md Pattern 4. Use the EXACT schema from the research file — do not abbreviate or modify. Key points to verify are present:

- `Venue` model: id (UUID), name, slug (@unique), logoUrl, paymentMode (enum), timestamps, relations to categories and orders
- `PaymentMode` enum: PREPAY_REQUIRED, PAY_AT_COUNTER, BOTH — with @default(BOTH) on Venue
- `MenuCategory` model: id (UUID), venueId (UUID FK), name, sortOrder, timestamps
- `MenuItem` model: id (UUID), venueId (UUID FK), categoryId (UUID FK), name, description (nullable), price (Decimal 10,2), imageUrl (nullable), isAvailable (@default(true)), sortOrder, **metadata Json @default("{}") @db.JsonB**, timestamps
- `Order` model: id (UUID), venueId (UUID FK), status (OrderStatus enum @default(PENDING_PAYMENT)), paymentMethod (PaymentMethod enum), paymentIntentId (nullable String), totalAmount (Decimal 10,2), referenceCode (@unique), timestamps
- `OrderStatus` enum: PENDING_PAYMENT, RECEIVED, PREPARING, READY, COMPLETED, CANCELLED
- `PaymentMethod` enum: STRIPE, PAY_AT_COUNTER
- `OrderItem` model: id (UUID), orderId (UUID FK), menuItemId (nullable UUID FK with onDelete: SetNull), **itemNameAtOrder String @map("item_name_at_order")** (non-nullable), **unitPriceAtOrder Decimal @db.Decimal(10,2) @map("unit_price_at_order")** (non-nullable), quantity Int, **selectedModifiers Json @default("[]") @db.JsonB @map("selected_modifiers")**, createdAt only (no updatedAt — snapshots are immutable)
- `StripeEvent` model: id (UUID), **stripeEventId String @unique @map("stripe_event_id")**, processedAt (@default(now())), eventType String

All models use `@@map` to snake_case table names. All IDs use `@id @default(uuid()) @db.Uuid`. Timestamps use `@map("created_at")` / `@map("updated_at")` conventions.

The schema must enforce:
- INFR-03: itemNameAtOrder and unitPriceAtOrder are non-nullable (no `?`) on OrderItem
- INFR-02: stripeEventId has @unique on StripeEvent
- JSONB extensibility: metadata on MenuItem, selectedModifiers on OrderItem, both with @db.JsonB
