**Step 1 — Create OrdersService (apps/api/src/orders/orders.service.ts):**

Install nanoid first (from apps/api/): `pnpm add nanoid`

Service methods:

```typescript
import { customAlphabet } from 'nanoid';
const generateRef = customAlphabet('0123456789ABCDEFGHIJKLMNOPQRSTUVWXYZ', 8);
```

`create(slug: string, dto: CreateOrderDto)`:
- Find venue by slug (throw NotFoundException if not found)
- Validate all menuItemIds exist and belong to this venue; fetch current prices for snapshot
- Calculate totalAmount from current item prices * quantities (INFR-03 snapshot requirement)
- If dto.paymentMethod === 'PAY_AT_COUNTER': create Order with status RECEIVED immediately
- If dto.paymentMethod === 'STRIPE': create Order with status PENDING_PAYMENT
- referenceCode = generateRef()
- Create OrderItems with itemNameAtOrder = item.name, unitPriceAtOrder = item.price (snapshots)
- Return: `{ id, referenceCode, status, totalAmount, venueSlug: slug }`

`createPaymentIntent(orderId: string)`:
- Find order, verify status is PENDING_PAYMENT (throw BadRequestException if not)
- Create Stripe PaymentIntent: `{ amount: Math.round(totalAmount * 100), currency: process.env.STRIPE_CURRENCY ?? 'gbp', automatic_payment_methods: { enabled: true }, metadata: { orderId } }`
- Update order: `prisma.order.update({ where: { id: orderId }, data: { paymentIntentId: paymentIntent.id } })`
- Return: `{ clientSecret: paymentIntent.client_secret }`

`markOrderReceived(orderId: string, paymentIntentId: string)`:
- `prisma.order.update({ where: { id: orderId }, data: { status: 'RECEIVED' } })`
- Log success

`cancelOrder(orderId: string, reason: string)`:
- `prisma.order.update({ where: { id: orderId }, data: { status: 'CANCELLED' } })`

`findPublic(orderId: string, slug: string)`:
- Find order with venue (verify venue.slug matches slug for security)
- Return: `{ id, referenceCode, status, customerName, totalAmount, createdAt, items: [{ itemNameAtOrder, unitPriceAtOrder, quantity }] }`

Interface `CreateOrderDto`:
```typescript
export interface CreateOrderDto {
  customerName: string;
  paymentMethod: 'STRIPE' | 'PAY_AT_COUNTER';
  items: Array<{ menuItemId: string; quantity: number }>;
}
```

Initialize Stripe in service: `private readonly stripe = new Stripe(process.env.STRIPE_SECRET_KEY ?? '')` (same pattern as webhook stub).

```typescript
switch (event.type) {
  case 'payment_intent.succeeded': {
    const paymentIntent = event.data.object as Stripe.PaymentIntent;
    const orderId = paymentIntent.metadata?.orderId;
    if (!orderId) {
      this.logger.warn(`payment_intent.succeeded missing orderId metadata: ${paymentIntent.id}`);
      break;
    }
    await this.ordersService.markOrderReceived(orderId, paymentIntent.id);
    this.logger.log(`Order ${orderId} marked RECEIVED via webhook`);
    break;
  }
  case 'payment_intent.payment_failed': {
    const paymentIntent = event.data.object as Stripe.PaymentIntent;
    const orderId = paymentIntent.metadata?.orderId;
    if (orderId) {
      await this.ordersService.cancelOrder(orderId, 'payment_failed');
      this.logger.log(`Order ${orderId} CANCELLED via payment_failed webhook`);
    }
    break;
  }
  default:
    this.logger.log(`Unhandled Stripe event: ${event.type}`);
}
```
